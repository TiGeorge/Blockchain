Find array elements in another array Open on JetBrains Academy

Apply binary search to solve the problem. You are given four lines as input:

  • the first line contains one integer \( n \) so that  \( 1 \le n \le 10^5 \);
  • the second one consists of an array \( A[1 \ldots n] \) of \( n \) various natural numbers, each not exceeding \( 10^9 \), in ascending order;
  • the third line contains one integer \( k \) so that \( 1 \le k \le 10^5 \);
  • the fourth one consists of \( k \) natural numbers \( b_1, \ldots, b_k \), each not exceeding \( 10^9 \).

Task:

For each \( i \) from 1 to \( k \) it is necessary to output index \( 1 \le j \le n \), so that \( A[j]=b_i \).

If there is no such \( j \), output -1.


Sample Input:
5
1 5 8 12 13
5
8 1 23 1 11
Sample Output:
3 1 -1 1 -1


Memory limit: 256 MB
Time limit: 3 seconds
